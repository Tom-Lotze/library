Tom's Library

Introduction

Welcome to my multimedia library. For now, the library is separated into books and films. You can add either books or films or series using URLs of either Bol.com or IMDb to automatically add the item to your library. This is done through scraping of the webpages using a BeautifulSoup4 based Python scraper. The instances are saved in a sqlite database, running in a Django framework.

Functionality

The functionality throughout the books and films library is practically the same. All instances have a status (e.g. Seen or On Watchlist for films) that can be changed on the detail pages of the instance. On the home page of each library various selections of the items can be seen, as well as a feed with the latest modifications in the database. Furthermore, one can add items here, or search in the respective library. Lastly, a short overview of statistics is shown on each home page, indicating how many pages have been read, or how many minutes of film has been watched.

Per item (book/film/series), one can see information about the item on the detail page, such as starring actors, author, length (time/ number of pages), release date, language, and the cover image. Furthermore, a description and metascores can be found here. At this point, you can change the status of the item (e.g. Currently Reading/Read etc.), place a personal review and a rating (0-5 stars), and optionally favorite the item. If necessary, the item can be removed from the library.
